# 干系人档案模板

- 类型：模板
- 适用阶段：需求分析 / 价值需求分析
- 来源：`sources/books/original/有效需求分析（第2版）.md`（第5章，表5-1）
- 摘要：识别出关键干系人后，还应进一步记录其基本信息、关注点、阻力点与重要度。干系人档案帮助团队把“谁会支持、谁会阻碍、为何支持/阻碍”写清楚。

## 输入条件
- 已形成干系人列表
- 需要对关键干系人展开定向调研或深度分析

## 关键步骤
1. 每张档案只针对一个关键干系人。
2. 填写基本信息：名称、类别、相关度、影响度、职责、代表、联系方式。
3. 记录核心关注点：正需求、阻力点/负需求、可选功能需求、重要度。
4. 补充备注信息，如专业背景、职业经历、沟通偏好等。
5. 对阻力点采用“避免××”等更可被客户接受的表述。

## 输出结果
- 可直接放入需求规格说明书的干系人分析条目
- 对调研、沟通、方案取舍更有指导性的干系人画像

## 适用边界
- 适用于关键干系人较多、诉求冲突明显的项目
- 干系人少且诉求简单时，可退化并入干系人列表说明栏

## 常见错误
- 只写关注点，不写阻力点
- 多个干系人混写在同一张档案中
- 阻力点写法过于刺激客户情绪
