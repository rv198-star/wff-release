# Stage-01 输出模板（审计镜像）— acceptance-coverage-planning

## 1. 这个文件是做什么的
- 这是 Stage-01 英文输出模板的中文审计镜像。
- 它帮助评审者理解：最终 validation planning package 长什么样、哪些字段必须显式表达 traceability、gate posture 与下游条件。

## 2. 最关键字段
- implementation handoff summary
- acceptance scope summary
- acceptance checklist reference
- coverage explanation reference
- gate / execution-control linkage
- `planned_test_ids`
- `stage02_may_start`

## 3. 交接要求
- 最终交给 Stage-02。
- handoff package 至少要包含：acceptance mapping、coverage rationale、gate posture、execution-control posture、Stage-02 may-start 决策。
