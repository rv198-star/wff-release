# 快速划分用户群

- 类型：方法
- 适用阶段：需求分析 / 产品定位 / 产品探索
- 来源：`sources/books/original/幕后产品_打造突破式产品思维.md`（第2章 第3节）
- 摘要：在产品早期，与其试图一次性理解“所有用户”，不如先通过有限访谈提炼关键因子，快速划分用户群，再通过问卷或其他方式验证这种分群是否成立。

## 输入条件
- 当前需要理解目标用户群、市场空白和竞争定位
- 还没有完整、稳定的用户分群模型

## 关键步骤
1. 通过定性访谈覆盖几类不同用户。
2. 从访谈中提炼划分用户群的关键因子。
3. 用因子快速构造用户群模型。
4. 通过问卷或其他低成本方法做初步验证。

## 输出结果
- 一版可验证的用户群划分模型
- 更清晰的目标用户与竞争策略判断

## 适用边界
- 适用于新产品、新领域、需要快速理解市场分层的场景
- 不适用于已经拥有成熟、长期稳定细分模型的高度成熟领域

## 常见错误
- 一开始就试图精确刻画全部用户
- 只访谈，不做因子提炼
- 先画用户分群图，再事后找理由验证

## 关联模板

- 用户群划分说明
